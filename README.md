
Project Overview
This project analyzes 1M+ US flight records to evaluate airline performance, flight delays, cancellations, airport operations, and travel trends**. The dashboard transforms raw flight data into meaningful business insights using PostgreSQL and Power BI.

Business Objective
- Analyze flight performance and operational efficiency.
- Track flight delays and cancellation trends.
- Identify top-performing airlines and airports.
- Generate actionable insights through KPI tracking and trend analysis.

Tools & Technologies
- PostgreSQL
- Power BI
- Power Query
- DAX
- SQL
- Data Modeling

Key KPIs
- Total Flights
- Cancelled Flights
- Cancellation Rate
- Average Arrival Delay
- Average Departure Delay

Dashboard Features
- KPI Cards
- Airline Performance Analysis
- Airport Performance Analysis
- Delay Analysis
- Cancellation Trend Analysis
- Interactive Filters & Slicers

Key Insights
- Identified airlines with the highest flight volume.
- Analyzed major causes of flight delays and cancellations.
- Evaluated airport operational performance.
- Generated insights to improve flight efficiency and service reliability.

Project Outcome
Developed an interactive dashboard that helps monitor airline performance, understand delay patterns, track cancellation trends, and support data-driven operational decision-making.
